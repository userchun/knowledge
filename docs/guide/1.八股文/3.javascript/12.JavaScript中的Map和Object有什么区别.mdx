# 12.JavaScript中的Map和Object有什么区别

Object是键值对的集合，键必须是字符串或Symbol类型，而Map是键值对的有序集合，键可以是任意类型。

| 特性     | Object                                                            | Map                                                                                                            |
| -------- | ----------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| 键的类型 | 只能是字符串或Symbol                                              | 可以是任意类型                                                                                                 |
| 意外的键 | 可能会有原型链上的意外键                                          | 没有原型链上的意外键                                                                                           |
| 有序性   | 属性存储是无序的                                                  | 有 ，按插入顺序排序                                                                                            |
| Size属性 | 没有                                                              | 有 ，键值对个数可以通过size获取                                                                                |
| 可迭代性 | Object.values() Object.keys() Object.entries() 对对象进行转换访问 | 是可迭代的 ，提供了一些方法，Map.forEach() Map.prototype.keys() Map.prototype.values() Map.prototype.entries() |
| 性能     | 对小量数据性能较好 ，频繁删除添加键值对性能好                     | 对大量数据性能较好                                                                                             |
